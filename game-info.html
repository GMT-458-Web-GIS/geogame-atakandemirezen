About the Game

ADAPTATION TO WORLD Game is a game based on correctly guessing the countries shown on the world map.

How to Play?

The aim of the game is to correctly guess the country given on the map that appears on the screen. You earn points for each correct answer. The goal is to have as many seconds as possible to reach 100 points, and to do this you must answer 10 questions in a row correctly within the time limit.

Game Modes

  • Normal Mode: 20 countries will be asked.
  • Hard Mode: 50 countries will be asked and the difficulty level will be higher.

Hints

  • Observe the countries carefully, geographical features can help you on the map.
  • You can use your time more efficiently by giving correct answers quickly..

Game Rule

If you answer incorrectly, the game ends!
